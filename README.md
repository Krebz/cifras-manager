# Cifras Manager

Aplicação web para gerenciamento pessoal de cifras musicais.

O projeto foi criado com foco em:

- execução musical em Missas Católicas;
- transposição de acordes;
- leitura rápida no celular;
- auto-scroll;
- organização de playlists.

---

## Stack

- React
- TypeScript
- Vite
- Mantine
- pnpm

---

## Funcionalidades atuais

- Renderização de cifras
- Parser de acordes
- Transposição de tom
- Auto-scroll
- Controle de velocidade

---

## Objetivo do projeto

Criar um gerenciador pessoal de cifras simples, rápido e funcional para uso durante execuções musicais.

---

```bash
pnpm install
```
