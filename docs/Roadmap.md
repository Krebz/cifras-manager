# Roadmap

## MVP - Versão 1.0

* Cadastro de músicas.
* Pesquisa.
* Visualização de cifras.
* Transposição de tom.
* Ajuste de fonte.
* Favoritos.
* Referência musical.
* Observações da música.

## Versão 1.1

- Playlists
- Reordenação de músicas
- Navegação horizontal entre músicas

## Versão 1.2

* Compartilhamento de playlists.
* Auto rolagem.
* Melhorias de usabilidade.

## Versão 2.0

* Importação de PDF/DOC/DOCX.
* Estatísticas de uso.
* Administração avançada.

## Versão 3.0

* Sincronização entre dispositivos.
* Backup em nuvem.
* Recursos colaborativos (opcional).
* Migração para PostgreSQL (somente se necessário).

## Versão futura

- Política de privacidade
- Gestão de dados pessoais
- Adequações adicionais à LGPD
