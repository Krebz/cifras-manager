# Decisões Arquiteturais

## 2026-05-28

### Banco de Dados

**Decisão**
Utilizar SQLite.

**Motivação**
- Projeto pessoal
- Baixa complexidade
- Fácil backup

**Alternativas Avaliadas**
- PostgreSQL
- IndexedDB

## 2026-05-29

### Estratégia Inicial do Projeto

Decisão:
Evoluir a aplicação atual ao invés de reiniciar do zero.

Motivação:
- Estrutura atual já atende boa parte dos requisitos.
- Evitar retrabalho.
- Preservar aprendizado acumulado.
- Evolução incremental mais segura.

Alternativas Avaliadas:
- Reescrita completa.
